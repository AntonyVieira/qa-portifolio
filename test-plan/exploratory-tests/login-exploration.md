
# Exploratory Testing - Login

## Cenário 1
Login sem inserir usuário ou senha.

Resultado esperado:
Sistema deve bloquear login e exibir mensagem de erro.

---

## Cenário 2
Inserir usuário válido com vários espaços no final.

Resultado esperado:
Sistema deve ignorar espaços extras ou exibir erro apropriado.

---

## Cenário 3
Inserir senha no campo de usuário e usuário no campo de senha.

Resultado esperado:
Sistema deve rejeitar login.
