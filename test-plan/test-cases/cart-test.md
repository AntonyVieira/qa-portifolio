
# Test Cases - Cart

## CT-01 Adicionar produto ao carrinho

Passos:
1. Acessar lista de produtos
2. Clicar em "Add to cart" em um produto
3. Abrir carrinho

Resultado esperado:
Produto deve aparecer no carrinho.


## CT-02 Adicionar vários produtos

Passos:
1. Adicionar 3 produtos diferentes
2. Abrir carrinho

Resultado esperado:
Todos os produtos devem aparecer no carrinho.


## CT-03 Remover produto do carrinho

Passos:
1. Adicionar produto ao carrinho
2. Abrir carrinho
3. Clicar em remover

Resultado esperado:
Produto deve ser removido.
