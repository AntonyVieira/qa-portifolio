
# Test Cases - Login

## CT-01 Login com credenciais válidas

Passos:
1. Acessar https://www.saucedemo.com
2. Inserir usuário válido
3. Inserir senha válida
4. Clicar em login

Resultado esperado:
Usuário deve acessar a página de produtos.


## CT-02 Login com senha inválida

Passos:
1. Acessar página de login
2. Inserir usuário válido
3. Inserir senha incorreta
4. Clicar em login

Resultado esperado:
Sistema deve exibir mensagem de erro.


## CT-03 Login com campos vazios

Passos:
1. Abrir página de login
2. Não preencher os campos
3. Clicar em login

Resultado esperado:
Sistema deve solicitar preenchimento dos campos.
